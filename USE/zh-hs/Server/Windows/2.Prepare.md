## Windows 的准备工作
> 更新时间 2022/5/17

1. 前往 Java8 下载地址
   [甲骨文官方 Java8 下载地址](https://www.java.com/zh-CN/download/manual.jsp)
   
2. 下拉找到 `Windows 脱机` 与  `Windows 脱机（64位）` 
3. 点击你想下载的版本

   > 如果你是32位的机器或只想使用最高4096的内存来开服，请点击 `Windows 脱机` 
   > 
   > 如果你是64位的机器，请点击 `Windows 脱机（64位）` 
   > 
   > 通常情况下，我们推荐使用 `Windows 脱机（64位）`

4. 安装 Java（一路下一步即可）
