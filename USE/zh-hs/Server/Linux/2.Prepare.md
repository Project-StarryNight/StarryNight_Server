## Linux 的准备工作
> 更新时间 2022/5/17
安装openjdk-8
debian

```bash
sudo apt update
sudo apt upgrade
sudo apt install openjdk-8-jdk
```

centen os 

```bash
sudo yum update
sudo yum install java-1.8.0-openjdk-devel
```

arch

```bash
pacman -Syyu
pacman -S jdk8-openjdk
```
